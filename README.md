# MyGuru OMR Flask

🧠 **MyGuru OMR Smart Marker** 是一个基于 Flask 的图像识别自动批改系统，专为教师设计。

---

## 📌 功能简介

- ✅ 上传标准答案（Skema），自动识别题数与答案（PDF、图片、Word）
- ✅ 上传学生答题卡图像，自动识别选项并进行评分
- ✅ 显示每位学生得分、正确题号、错误题号
- ✅ 导出成绩为 Excel 表
- ✅ 内置模板支持 SPM / UPSR / SMK / SJKT / UASA 格式
- ✅ 提供网页服务接口，可整合到 Android App 上架 Google Play

---
